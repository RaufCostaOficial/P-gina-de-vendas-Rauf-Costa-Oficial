# 🚀 Página de Vendas - História da Paraíba

Este projeto contém a sua página de vendas altamente persuasiva, desenvolvida com **HTML5** e **Tailwind CSS**. Ela foi projetada para ser rápida, responsiva e focada em conversão para os seus infoprodutos.

## 📂 Estrutura do Projeto

- `index.html`: O arquivo principal que contém todo o código da página (estrutura, estilo e links).

## 💻 Como abrir no Visual Studio Code (VS Code)

1.  **Abra o VS Code.**
2.  Vá em `Arquivo` > `Abrir Pasta...` (ou `File` > `Open Folder...`).
3.  Selecione a pasta onde você descompactou este projeto.
4.  Para visualizar a página enquanto edita, recomendo instalar a extensão **"Live Server"** no VS Code. Após instalar, basta clicar com o botão direito no arquivo `index.html` e selecionar **"Open with Live Server"**.

## 🛠️ Como fazer edições simples

- **Mudar Links:** Procure por `href="https://go.hotmart.com/..."` para alterar os links da Hotmart ou `href="https://wa.me/..."` para o WhatsApp.
- **Alterar Textos:** Basta procurar o texto que deseja mudar entre as tags HTML (ex: `<h1>...</h1>` ou `<p>...</p>`).
- **Cores:** A página utiliza classes do Tailwind CSS (ex: `bg-gold`, `text-gray-900`). Você pode consultar a documentação oficial em [tailwindcss.com](https://tailwindcss.com).

## 🌐 Como colocar no seu Domínio

1.  Acesse o painel da sua hospedagem (ex: cPanel).
2.  Vá em **Gerenciador de Arquivos** > pasta `public_html`.
3.  Faça o upload do arquivo `index.html`.
4.  Se quiser que a página fique em uma subpasta (ex: `seudominio.com.br/historia`), crie uma pasta chamada `historia` dentro de `public_html` e coloque o arquivo lá.
